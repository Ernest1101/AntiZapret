# 🛡️ AntiZapret

Современный GUI-клиент для обхода DPI блокировок на базе **zapret** от [bol-van](https://github.com/bol-van/zapret-win-bundle).

> Вдохновлено проектом [Flowseal/zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube)

![Python](https://img.shields.io/badge/Python-3.8+-blue?logo=python)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey?logo=windows)
![License](https://img.shields.io/badge/License-MIT-green)

## ✨ Возможности

- 🎨 **Современный GUI** на базе CustomTkinter с тёмной темой
- 🔐 **Автозапрос прав администратора** при запуске
- ⚡ **Готовые стратегии** обхода блокировок
- ⚙️ **Управление службой Windows** с автозапуском
- 📋 **Редактор списков доменов** прямо в интерфейсе
- 📝 **Журнал событий** в реальном времени
- 🔄 **Автоматический мониторинг** статуса запущенных стратегий

## 📸 Интерфейс

Приложение имеет боковую панель навигации с разделами:
- 🏠 **Главная** — обзор состояния и быстрые действия
- ⚡ **Стратегии** — выбор и запуск стратегий обхода
- ⚙️ **Службы** — управление Windows-службой для автозапуска
- 📋 **Списки доменов** — редактирование списков заблокированных сайтов
- 📝 **Логи** — журнал событий программы
- ℹ️ **О программе** — информация и компоненты

## 🚀 Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/Ernest1101/AntiZapret.git
cd AntiZapret
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Скачивание бинарных файлов zapret

Скачайте последний релиз **zapret-win-bundle**:
🔗 https://github.com/bol-van/zapret-win-bundle/releases

Распакуйте и скопируйте содержимое папки `zapret-winws/` в папку `bin/` вашего репозитория.

**Обязательные файлы в `bin/`:**
| Файл | Назначение |
|------|-----------|
| `winws.exe` | ⭐ Главный исполняемый файл zapret |
| `WinDivert.dll` | DLL драйвера WinDivert |
| `WinDivert64.sys` | Драйвер WinDivert (64-bit) |
| `WinDivert32.sys` | Драйвер WinDivert (32-bit, опционально) |
| `cygwin1.dll` | Библиотека Cygwin |

## ▶️ Запуск

Просто запустите файл:

```bash
python gui.py
```

**Важно:** Программа **автоматически запросит права администратора** через UAC при запуске. Это необходимо для работы драйвера WinDivert и управления сетевыми фильтрами.

Если вы хотите собрать `.exe` файл:

```bash
pyinstaller --onefile --windowed --uac-admin --name=AntiZapret gui.py
```

## 📖 Использование

### Запуск стратегии

1. Перейдите во вкладку **⚡ Стратегии**
2. Выберите нужную стратегию из списка
3. Нажмите **▶ Запустить выбранную стратегию**
4. Индикатор в шапке станет зелёным — обход работает!

### Установка службы (автозапуск)

Если вы хотите, чтобы обход работал автоматически при старте Windows:

1. Перейдите во вкладку **⚙️ Службы**
2. Нажмите **📥 Установить службу**
3. Служба будет запускаться при каждом старте системы

### Редактирование списков доменов

1. Откройте вкладку **📋 Списки доменов**
2. Выберите нужный файл из выпадающего списка
3. Отредактируйте список (каждый домен/IP с новой строки)
4. Нажмите **💾 Сохранить**

## ⚠️ Важные замечания

1. **Требуются права администратора** — для работы драйвера WinDivert
2. **Антивирусы могут ругаться** — WinDivert использует низкоуровневый доступ к сети
3. **Windows 10/11** — рекомендуется использовать последние версии
4. **Не совместим с некоторыми VPN** — могут быть конфликты
