# AntiZapret GUI

Графический интерфейс для управления обходом блокировок DPI (Deep Packet Inspection).

## Описание

Это GUI-приложение для удобного управления стратегиями обхода блокировок Discord, YouTube и других сервисов. Основано на принципах работы zapret.

## Возможности

- 🎨 Удобный графический интерфейс
- 🚀 Запуск различных стратегий обхода
- ⚙️ Управление службами (установка/удаление)
- 📋 Редактирование списков доменов
- 🔄 Проверка статуса обхода
- 🎮 Игровой режим для UDP трафика

## Требования

- Windows 10/11
- Python 3.8+
- Бинарные файлы zapret (winws.exe, WinDivert)

## Установка

1. Скачайте или клонируйте репозиторий
2. Установите зависимости:
```bash
pip install -r requirements.txt
```
3. Поместите бинарные файлы zapret в папку `bin/`
4. Запустите приложение:
```bash
python gui.py
```

## Использование

### Первый запуск

1. Запустите `gui.py` от имени администратора
2. Выберите стратегию из списка
3. Нажмите "Запустить"
4. Проверьте работоспособность в браузере

### Доступные стратегии

- **General** - базовая стратегия
- **ALT (1-12)** - альтернативные стратегии с разными параметрами
- **FAKE TLS** - стратегии с подменой TLS handshake
- **SIMPLE FAKE** - упрощенные fake стратегии

### Службы

- **Install Service** - установить выбранную стратегию в автозапуск
- **Remove Services** - удалить все службы AntiZapret
- **Check Status** - проверить текущий статус

### Списки доменов

Вы можете редактировать списки доменов через меню:
- `lists/list-general.txt` - основные домены для обхода
- `lists/list-general-user.txt` - пользовательские домены
- `lists/ipset-all.txt` - IP адреса и подсети

## Важные замечания

### Антивирусы

WinDivert может вызвать реакцию антивируса. Это нормальное поведение, так как WinDivert - это инструмент для перехвата и фильтрации трафика. Добавьте папку с программой в исключения антивируса.

### Secure DNS

Для корректной работы рекомендуется включить Secure DNS:
- Chrome: Настройки → Конфиденциальность → Безопасность → Использовать безопасный DNS
- Firefox: Настройки → Основные → Сетевые настройки → Включить DNS через HTTPS

### Стратегии могут переставать работать

Стратегии обхода могут переставать работать со временем из-за обнаружения. Пробуйте разные стратегии, пока не найдете рабочую.
