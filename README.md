# 🛡️ AntiZapret

Современный GUI интерфейс для обхода DPI блокировок Discord, YouTube и других сервисов на базе zapret.

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey)
![Python](https://img.shields.io/badge/python-3.8+-green)

## ✨ Возможности

### 🎨 Современный интерфейс
- **CustomTkinter GUI** с тёмной/светлой темой
- **Боковая навигация** с 7 разделами
- **Карточки статуса** в реальном времени
- **Индикатор работы** с мониторингом uptime

### ⚡ 20+ стратегий обхода
- **General** - базовая стратегия
- **General (ALT)** - альтернативная (часто работает лучше)
- **General (ALT2-ALT12)** - 11 вариантов с разными параметрами
- **FAKE TLS AUTO** - автоматический fake TLS обход
- **FAKE TLS AUTO ALT/ALT2/ALT3** - варианты FAKE TLS
- **Simple Fake** - простой fake без split
- **Simple Fake ALT/ALT2** - варианты simple fake

### 🔍 Авто-тестирование
- **Проверка всех стратегий** одной кнопкой
- **Тестирование отдельной стратегии** с замером latency
- **Автоматический выбор** рабочей стратегии
- **Статистика успешных тестов**

### ⚙️ Управление службами
- Создание/удаление Windows службы
- Автозапуск при старте системы
- Мониторинг статуса службы

### 📋 Редактор списков
- Встроенный редактор доменов
- Вкладки для разных списков (Основные, Пользовательские, Google, IPSet)
- Сохранение/загрузка списков

### 📝 Система логов
- Журнал событий в реальном времени
- Экспорт логов в файл
- Цветовая разметка (OK/ERROR/INFO)

### 🔧 Настройки
- **Темы**: Светлая / Тёмная / Системная
- **Автозапуск** при старте Windows
- **Сворачивание в трей** при закрытии
- **Уведомления** о событиях
- **Экспорт/Импорт** конфигурации
- **Сброс статистики**

### ⌨️ Горячие клавиши
- **F5** - Запустить текущую стратегию
- **F6** - Остановить все процессы
- **F7** - Проверить все стратегии

### 📊 Статистика
- Всего запусков
- Успешных тестов
- Любимая стратегия
- Время работы

## 🚀 Установка

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/Ernest1101/AntiZapret.git
cd AntiZapret
```

### 2. Установите зависимости
```bash
pip install -r requirements.txt
```

### 3. Скачайте бинарные файлы
Скачайте **zapret-win-bundle** с [GitHub](https://github.com/bol-van/zapret-win-bundle/releases) и скопируйте содержимое папки `zapret-winws` в папку `bin/`:

- ✅ `winws.exe` - главный файл zapret
- ✅ `WinDivert.dll` - DLL драйвера
- ✅ `WinDivert64.sys` - драйвер (64-bit)
- ✅ `cygwin1.dll` - библиотека Cygwin
- ✅ Другие файлы из `zapret-winws`

### 4. Запустите GUI
```bash
python gui.py
```

При запуске автоматически запросит права администратора через UAC.

## 📖 Использование

### Быстрый старт
1. Запустите `python gui.py` (от имени администратора)
2. Перейдите на вкладку **"⚡ Стратегии"**
3. Выберите стратегию (рекомендуем начать с **General (ALT)**)
4. Нажмите **"▶️ Запустить"**
5. Проверьте работу Discord/YouTube

### Авто-тест стратегий
1. Перейдите на вкладку **"🔧 Настройки"**
2. Укажите URL для теста (по умолчанию `https://discord.com`)
3. Нажмите **"🔍 Проверить все стратегии"**
4. Программа автоматически протестирует все 20+ стратегий
5. В логах появится список рабочих стратегий с latency

### Установка в автозапуск
1. Выберите нужную стратегию
2. Перейдите на вкладку **"⚙️ Службы"**
3. Нажмите **"Создать службу"**
4. Стратегия будет запускаться автоматически при старте Windows

## 🔧 Настройка стратегий

Все стратегии находятся в `core/strategy.py`. Каждая стратегия содержит:

- **name** - отображаемое имя
- **description** - описание
- **params** - параметры для winws.exe

### Пример стратегии:
```python
"general_alt": {
    "name": "General (ALT)",
    "description": "Альтернативная стратегия - часто работает лучше",
    "params": [
        "--wf-tcp=80,443,2053,2083,2087,2096,8443",
        "--wf-udp=443,19294-19344,50000-50100",
        "--filter-l7=discord,youtube,stun,quic",
        "--hostlist-auto=lists\\list-general.txt",
        "--dpi-desync=fake,fakedsplit",
        "--dpi-desync-fooling=ts",
        "--dpi-desync-fake-discord=bin\\files\\quic_initial_dbankcloud_ru.bin",
        "--dpi-desync-fake-tls=bin\\files\\tls_clienthello_4pda_to.bin"
    ]
}
```

## ⚠️ Важная информация

### Требования
- Windows 10/11 (64-bit)
- Python 3.8+
- Права администратора
- Бинарные файлы zapret-win-bundle

### Антивирусы
WinDivert может вызвать реакцию антивируса. Это нормальное поведение, так как WinDivert - инструмент для перехвата трафика. Добавьте папку `bin/` в исключения антивируса.

### Secure DNS
Для корректной работы рекомендуется включить Secure DNS:
- **Chrome**: Настройки → Конфиденциальность → Использовать безопасный DNS
- **Firefox**: Настройки → Сетевые настройки → DNS через HTTPS
- **Windows 11**: Параметры → Сеть и Интернет → DNS

## 🐛 Решение проблем

### Discord не заходит
1. Попробуйте разные стратегии (ALT, FAKE TLS)
2. Используйте авто-тест для поиска рабочей стратегии
3. Отключите Secure DNS в настройках Discord
4. Перезапустите Discord после запуска стратегии

### YouTube не работает
1. Убедитесь что включён Secure DNS в браузере
2. Отключите блокировщик рекламы
3. Попробуйте разные стратегии

### winws.exe не запускается
1. Проверьте наличие всех файлов в папке `bin/`
2. Запустите от имени администратора
3. Проверьте логи на вкладке "📝 Логи"
