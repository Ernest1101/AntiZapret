# Папка для бинарных файлов

Сюда нужно поместить следующие файлы из [zapret-win-bundle](https://github.com/bol-van/zapret-win-bundle):

## Необходимые файлы:

- **`winws.exe`** - Основной исполняемый файл zapret
- **`WinDivert.dll`** - Библиотека WinDivert
- **`WinDivert64.sys`** - Драйвер WinDivert для 64-битных систем

## Где взять:

1. Скачайте последний релиз с https://github.com/bol-van/zapret-win-bundle/releases
2. Распакуйте архив
3. Скопируйте указанные файлы в эту папку (`bin/`)

## Структура:

```
bin/
├── winws.exe
├── WinDivert.dll
├── WinDivert64.sys
└── README.md
```

**Важно:** Без этих файлов программа не сможет работать!
